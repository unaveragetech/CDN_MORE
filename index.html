Comprehensive Guide to Content Delivery Networks (CDNs)

Content Delivery Networks (CDNs) are systems of distributed servers that deliver web content to users based on their geographic location. Using a CDN can significantly improve load times, reliability, and performance of your website. Below is a list of popular CDNs along with code examples for various types of resources.

Popular CDNs

  • CDNJS - A popular library for hosting JavaScript libraries and other assets.
  • jsDelivr - A free CDN for open-source projects.
  • Google Hosted Libraries - A collection of popular JavaScript libraries.
  • UNPKG - A fast CDN for everything on npm.
  • Microsoft Ajax CDN - CDN for hosting JavaScript libraries and other resources.
  • Bootstrap CDN - A CDN specifically for Bootstrap and its dependencies.
  • Cloudflare CDN - A general-purpose CDN for hosting various assets.

Comprehensive Lists and Resources

  • CDNJS Directory - A comprehensive directory of libraries hosted on CDNJS.
  • GitHub CDN - List of projects hosted on GitHub accessible via CDN.
  • Awesome CDN - A curated list of various CDNs on GitHub.
  • Public CDN List - A list of various public CDNs.

Special Purpose CDNs

  • FontCDN - A CDN specifically for web fonts.
  • Cloudflare Fonts - A service from Cloudflare for hosting Google Fonts.

Code Examples

1. Embedding jQuery from CDNJS

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>

<script src="https://cdnjs.cloudflare.com/ajax/libs/jquery/3.6.0/jquery.min.js"></script>
